How to Update the Ward List Data

This guide explains how to download the current ward list from the Church website, clean it, and upload it here.

Phase 1: Download Data from LCR

  1. Log in to LCR (Leader and Clerk Resources).
  2. Go to Membership → Member List.
  3. Click the Print/Export button (usually a printer icon).
  4. Choose Export to CSV.
    (Note: Do NOT choose PDF. We need the data file.)
  5. Save the file to your computer. It is usually named Member List.csv.

Phase 2: Clean the File

The raw file contains "junk" headers (like Ward Name and Date) that must be removed before uploading.

Option A: Manual Cleaning (Excel/Numbers)

  1. Open the CSV file in Excel or Numbers.
  2. Highlight the top rows containing the Ward Name/Date.
  3. Delete these rows entirely.
  4. Ensure the very first row (Row 1) now contains the column headers (e.g., Preferred Name, Household Position).
  5. Save/Export the file as CSV.

Option B: Using the Script (Admin)

If you have the python helper script:

python3 clean_ward_list.py "path/to/Member List.csv"

Phase 3: Upload to the Quiz

  1. Go back to the Home Page.
  2. Click Choose File.
  3. Select your cleaned CSV file.
  4. Click Start Quiz.
⚠️ Security Note: This file contains real member contact information. Please delete the CSV files from your computer immediately after uploading.
